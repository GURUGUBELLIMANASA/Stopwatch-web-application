# ⏱️ Stopwatch Web Application

## 📌 Project Overview
This project is a simple and interactive Stopwatch Web Application built using HTML, CSS, and JavaScript.  
It allows users to start, pause, reset, and track elapsed time accurately.

---

## 🚀 Features
- ▶️ Start the stopwatch
- ⏸️ Pause/Stop the stopwatch
- 🔄 Reset the timer
- ⏱️ Real-time time tracking (hours, minutes, seconds)
- 📱 Responsive design for all devices

---

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 💡 How It Works
- JavaScript is used to track time using intervals.
- The `setInterval()` function updates the timer every second.
- Start button begins the timer.
- Pause button stops the timer.
- Reset button clears the timer and sets it back to zero.
